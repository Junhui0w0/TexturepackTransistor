# Miecraft_TexturepackTransistor

-This is transistor which can transfer 1.8(or High) texturepack to 1.5.2  

-1.8 혹은 그 이상의 마인크래프트 버전 텍스쳐팩을 1.5.2로 전환시킬 수 있는 프로그램입니다.
